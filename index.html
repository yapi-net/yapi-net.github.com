Yandex.Direct API library for .NET

What is the library?

The library provides strong-typed and modern .NET interface to the JSON-based Yandex.Direct API with support of all available authentication scenarios.

Do you have some idea or know about a problem? – you're welcome to give us feedback!
Do you want to contribute and make the library better? – you're welcome to join the project!

How to get it?

The library can be found on NuGet Gallery: Yandex.Direct

Sources available at: https://github.com/yapi-net/YandexDirect

We're still polishing the way how the library can be used from your code and we haven't finished yet, so most current version of the library can taken directly from the source code, NuGet releases are outdated a bit.

How to report a problem?

Bug tracker: https://github.com/yapi-net/YandexDirect/issues
